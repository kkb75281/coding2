Layout

레이아웃이란 이미지나 글 등을 화면에서 보기 좋게 배열하는 것을 말합니다. 이번 게시글에서는 layout을 만드는 3가지 방법을 알아보려고 합니다.

1. Float-based layout

float의 사전적 의미는 '뜨다','떠돌다'이며 말 그대로 특정 요소를 뜨게 하도록 해주는 속성입니다. 플로팅 된 요소는 블럭 요소가 되며 display: inline-block;을 한 것과 같이 됩니다.

01. float: 속성값(value);

left; 요소를 왼쪽 방향으로 띄운다
right; 요소를 오른쪽 방향으로 띄운다
none; 기본값, 요소를 띄우지 않는다

02. 플로팅 해제

플로팅한 요소는 기본적인 문서의 배치에서 벗어난 상태이기 때문에 레이아웃이 깨지는 버그가 발생합니다. 이때 플로팅을 해제하는 방법 3가지를 알아보겠습니다.

clear; 임시로 플로팅 되는 것을 막기 위해 나온 속성
overflow; 내가 지정한 영역 외에는 안보이게 해주는 속성, height 값이 없어지는 버그는 사라지지만 2차메뉴, 3차메뉴 만들게되면 쓰지 못한다.
clearfix; 기본값, 요소를 띄우지 않는다

See the Pen if문 by kkb75281 (@kkb75281) on CodePen.

2. Flex-based layout

하나의 실행문이 끝나면 마지막에 세미콜론(;)을 붙여 줍니다. 세미콜론(;)을 붙여주지 않아도 상관은 없습니다.

var num = 5;
document.write(num);

2. 대소문자

자바스크립트 명령어는 대소문자를 구별합니다. console과 Console은 각각 다른 문자열로 처리됩니다.

3. 들여쓰기

자바스크립트는 가독성을 위하여 들여쓰기를 사용합니다.

4. 주석

자바스크립트 주석은 일반적으로 특정 코드에 대한 설명을 입력하거나 특정한 부분의 코드가 실행되지 않도록 사용합니다. 주석 표시는 프로그램에 영향을 미치지 않습니다.

출력하는 방법

자바스크립트에서는 출력할 수 있는 방법이 여러가지가 있습니다.

분류 메서드
입력 prompt()

자바스크립트 사용 방법

자바스크립트의 사용 방법은 1. 외부파일로 연결하는 방법 2. 스크립트 태그에 사용하는 방법 3. 태그에 직접 기술하는 방법 등이 있습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Javasript</title>
    <!-- 1. 외부 파일로 로드하는 방법 -->
    <script type="text/Javasript" src="js/script.js"></script>
    <!-- 2. script 태그 사이에 기술하는 방법 -->
    <script>
        document.write("Javasript");
    </script>
</head>
<body>
    <!-- 3. 태그에 직접 기술하는 방법 -->
    <a href="#" onclick="alert('Javasript')"></a>
</body>
</html>
