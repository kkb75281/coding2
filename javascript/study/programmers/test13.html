문자열 내 p와 y의 개수 13. 대문자와 소문자가 섞여있는 문자열 s가 주어집니다. s에 'p'의 개수와 'y'의 개수를 비교해 같으면 True, 다르면 False를 return 하는 solution를 완성하세요. 'p', 'y' 모두 하나도 없는 경우는 항상 True를 리턴합니다. 단, 개수를 비교할 때 대문자와 소문자는 구별하지 않습니다. 예를 들어 s가 "pPoooyY"면 true를 return하고 "Pyy"라면 false를 return합니다. * 문자열 s의 길이 : 50 이하의 자연수
* 문자열 s는 알파벳으로만 이루어져 있습니다. 
function solution(s){
    let text = s.toLowerCase();
    let textP = text.split("p").length;
    let textY = text.split("y").length;
    
    return textP == textY ? true : false;
}
나의 풀이 대소문자가 섞여 있을 수 있기 때문에 전부 소문자로 바꿔준 후 p와 y 문자열의 갯수를 비교해주었습니다. 
function numPY(s){
    return s.toUpperCase().split("P").length === s.toUpperCase().split("Y").length;
}
다른 풀이 제가 한 풀이를 간단하게 삼항연산식으로 바꿔준 풀이네요 ! 
function numPY(s) {
    return s.match(/p/ig).length == s.match(/y/ig).length;
}
