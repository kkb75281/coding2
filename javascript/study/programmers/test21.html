내적 21. 길이가 같은 두 1차원 정수 배열 a, b가 매개변수로 주어집니다. a와 b의 내적을 return 하도록 solution 함수를 완성해주세요. 이때, a와 b의 내적은 a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1] 입니다. (n은 a, b의 길이) * a, b의 길이는 1 이상 1,000 이하입니다.
* a, b의 모든 수는 -1,000 이상 1,000 이하입니다. 
function solution(a, b) {
    var answer = 0;
    
    for (let i in a){
        answer += a[i] * b[i]
    }
    
    return answer;
}
나의 풀이 각 배열의 같은 인덱스 값을 for문을 활용하여 더하고 answer 값에 추가시켰습니다. 
function solution(a, b) {
    return a.reduce((acc, _, i) => acc += a[i] * b[i], 0);
}
다른 풀이 Reduce를 활용하여 만들어줬네요 !
