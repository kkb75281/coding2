* s의 맨앞에는 부호(+, -)가 올 수 있습니다.
* s는 부호와 숫자로만 이루어져있습니다.
* s는 "0"으로 시작하지 않습니다.
ex) str = "1234" => 1234 반환 / str = "-1234" => -1234 반환 
function solution(s) {
    return Number(s);
}
