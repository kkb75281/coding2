문자열 내림차순으로 배치하기 26. 문자열 s에 나타나는 문자를 큰것부터 작은 순으로 정렬해 새로운 문자열을 리턴하는 함수, solution을 완성해주세요. s는 영문 대소문자로만 구성되어 있으며, 대문자는 소문자보다 작은 것으로 간주합니다. * str은 길이 1 이상인 문자열입니다. 
function solution(s) {
    return s.split('').sort().reverse().join('');
}
나의 풀이 문자열을 split 메서드를 통해 배열로 쪼개기 - sort와 reverse로 내림차순으로 정렬시키기 - join을 사용하여 문자열로 변환 
function solution(s) {
    return s.split("").sort((a,b) => a<b ? 1:-1).join("")
}
다른 풀이 삼항연산식을 사용해서 풀어주었네요 !
