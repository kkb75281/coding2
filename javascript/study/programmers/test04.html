수박수박수박수박수박수? 04. 길이가 n이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요. 예를들어 n이 4이면 "수박수박"을 리턴하고 3이라면 "수박수"를 리턴하면 됩니다. * n은 길이 10,000이하인 자연수입니다.
ex) n=3 => return = "수박수", n=4 => return = "수박수박" 
function solution(n) {
    var answer = '';
    for(i=1; i<=n; i++){
        if(i%2==0){
            answer += "박";
        } else {
            answer += "수";
        }
    }
    return answer;
}
나의 풀이 for문을 활용하여 홀수번째에는 "수"를, 짝수번째에는 "박"을 answer에 추가시켰습니다.
