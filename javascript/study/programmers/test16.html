하샤드 수 16. 양의 정수 x가 하샤드 수이려면 x의 자릿수의 합으로 x가 나누어져야 합니다. 예를 들어 18의 자릿수 합은 1+8=9이고, 18은 9로 나누어 떨어지므로 18은 하샤드 수입니다. 자연수 x를 입력받아 x가 하샤드 수인지 아닌지 검사하는 함수, solution을 완성해주세요. * x는 1 이상, 10000 이하인 정수입니다. 
function solution(x) {
    var sum = 0;
    var arr = String(x).split("");
    
    for(let i=0; i<arr.length; i++){
        sum += Number(arr[i])
    }
    return (x % sum == 0) ? true:false;
}
나의 풀이 split 메서드를 사용해 자릿수대로 나눠준 후 각 숫자를 더해 삼항연산식으로 판별해주었습니다.
